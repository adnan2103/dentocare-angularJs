• As a dento care admin :
  • I can setup Clinic Account as ClinicAdmin
  • I can see list of such clinic's account created with basic information.
  • In clinic detail page I can see the modules what clinic have purchased and when they are expiring.
  • I can add more modules or extend expiry by adding payment for a clinic.
  • I can create new clinic users ClinicUser and give access to purchased modules.
  • By default ClinicAdmin will see all the links in application but can access to only purchased modules and non expired, for non purchased modules or expired modules he will get a message to purchase it. ClinicUser will see or have access only to the modules for which access is given and not expired.
  • I can see old payment made with history.
